# 🍪 CookieGrab

A lightweight Chrome extension to export session cookies from Instagram and Facebook.

---

## Features

- One-tap cookie generation
- Login session detection
- Export as JSON file
- Copy to clipboard

---

## Installation

1. Download the latest release zip from [Releases](../../releases)
2. Extract the zip
3. Open Chrome → `chrome://extensions`
4. Enable **Developer Mode**
5. Click **Load unpacked** → select the extracted folder
